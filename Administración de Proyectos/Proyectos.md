# Proyectos

**Proyecto**: Un esfuerzo temporal que se lleva a cabo para crear un producto,
servicio o resultado único.

| Proyecto                                  | Operación                       |
| ----------------------------------------- | ------------------------------- |
| Temporal (inicio y fin definidos).        | Continúa (sin fin determinado). |
| Crea algo nuevo.                          | Mantiene el funcionamiento.     |
| Equipo interfuncional, temporal y ad-hoc. | Equipo funcional y permanente.  |
| Usos presupuestos de inversión.           | Gasto operativo.                |

## Proyecto vs Operación

- Las operaciones identifican las necesidades de un proyecto.
- En respuesta a esta necesidad, se inicia un proyecto para construir o adquirir
  una nueva solución.
- Al finalizar el proyecto se entrega a operaciones para su mantenimiento.

## Roles dentro de un Proyecto

- **Project Sponsor**: Ejecutivo Senior con autoridad financiera. Tiene la
  autoridad política para resolver conflictos que exceden al Project Manager. El
  es el que establece prioridades entre proyectos. Es el máximo responsable de
  defender el proyecto ante la organización.
- **Project Manager**: Responsable qe se alcancen los objetivos (alcance,
  presupuesto y cronograma). Se dedica a gestionar las restricciones del
  proyecto y liderar al equipo. Debe entender por qué el proyecto es importante
  para la organización para tomar las decisiones que se alineen al valor
  comercial.
- **Equipo de Trabajo**: Individuos que proucen los entregables. Reportan a su
  jefe inmediato y al gestor de proyectos.
- **Stackeholders**: Cualquier individuo que puede afectar o verse afectado por
  el proyecto. Deben ser incluidos durante la gestión del proyecto para toma de
  decisiones. Incluso pueden tener tanto poder como para frenar la ejecución del
  proyecto.
